nineku.fm
=================

###介绍：

----------------------------

http://www.9ku.com/fm的终端版本



###安装:

---------------------------------------

npm install -g nineku.fm
windows64位下安装编译会出现问题，需要在cmd中更改成64位编译模式，请参考：
https://msdn.microsoft.com/zh-cn/library/x4d2c09s.aspx
后运行安装命令

###运行:

---------------------------------

nineku.fm
